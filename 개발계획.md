
### 개발 기능
3. 홈 대시보드 (신규)
오늘 하루를 한눈에 파악할 수 있는 통합 뷰

오늘의 할 일 요약 (완료 N / 전체 N)
오늘 체크할 습관 목록
현재 스트릭(연속 달성일) 표시
주간 달성률 미니 그래프


4. To-Do 기능
✅ 할 일 관리

할 일 추가 / 수정 / 삭제
완료 체크 및 취소선 애니메이션
중요도 설정 (높음 / 보통 / 낮음)
마감일 설정

📁 카테고리 & 태그

카테고리별 분류 (업무, 개인, 쇼핑 등)
커스텀 태그 추가

🔍 필터 & 정렬

날짜 / 우선순위 / 카테고리 필터
완료 / 미완료 토글


5. Habit Tracker 기능 (신규)
🔥 습관 관리

습관 추가 / 수정 / 삭제
반복 주기 설정

매일 / 특정 요일 / N일마다


목표 횟수 설정 (예: 하루 2회 물 마시기)
카테고리 분류 (건강, 공부, 운동 등)
아이콘 & 색상 커스터마이징

📅 체크인

오늘의 습관 목록에서 원터치 체크
부분 완료 지원 (3회 중 2회 완료)
날짜별 되돌아보기 (캘린더 뷰)

🏆 스트릭 & 동기부여

연속 달성일(Streak) 표시 🔥
최장 스트릭 기록 보관
달성 시 간단한 축하 애니메이션
주간 / 월간 완료율 표시


6. 통계 화면 (신규)
섹션내용To-Do 리포트주간/월간 완료 수, 카테고리별 분포Habit 리포트습관별 달성률, 스트릭 히스토리히트맵GitHub 잔디처럼 날짜별 활동량 시각화인사이트"이번 주 가장 잘 지킨 습관: 운동 💪"


### 화면 구성(전체)
📱 앱 구조
├── 홈 (대시보드)
│   ├── 오늘의 To-Do 요약
│   ├── 오늘의 습관 체크리스트
│   └── 주간 미니 그래프
│
├── To-Do
│   ├── 할 일 목록 (필터/정렬)
│   ├── 할 일 상세/편집
│   └── 카테고리 관리
│
├── Habit
│   ├── 습관 목록
│   ├── 습관 추가/편집
│   ├── 캘린더 뷰 (체크 이력)
│   └── 스트릭 현황
│
├── 통계
│   ├── To-Do 리포트
│   ├── Habit 리포트
│   └── 히트맵
│
└── 설정
    ├── 다크모드
    ├── 알림 설정
    └── 데이터 백업/내보내기


### 전체 아키텍쳐
사용자 (PC / 스마트폰 / 태블릿)
        ↓  브라우저 접속 (URL 하나로 모든 기기)
   [프론트엔드]  ←→  [백엔드 API]  ←→  [데이터베이스]
    React 앱        서버 로직         데이터 저장
        ↓
   [인증 서비스]
    로그인/회원가입

### 기술 스택
✅ 확정 기술 스택
영역                 기술                            비용
프론트엔드    React + TypeScript + Tailwind CSS      무료
인증         Supabase Auth (이메일 + 소셜)            무료
데이터베이스  Supabase PostgreSQL                    무료
실시간 동기화 Supabase Realtime                      무료
배포         Vercel                                 무료
PWA         Vite PWA Plugin                         무료



### DB테이블 설계

-- 사용자 (Supabase Auth가 자동 생성)
users

-- 할 일
todos
├── id
├── user_id
├── title
├── memo
├── category
├── priority     -- high / medium / low
├── due_date
├── is_completed
└── created_at

-- 습관
habits
├── id
├── user_id
├── title
├── category
├── icon
├── color
├── repeat_type  -- daily / weekly / custom
├── repeat_days  -- [mon, wed, fri] 등
├── goal_count   -- 하루 목표 횟수
└── created_at

-- 습관 체크 기록
habit_logs
├── id
├── habit_id
├── user_id
├── date
├── count        -- 완료 횟수
└── created_at
